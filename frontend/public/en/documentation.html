🏠 1. Installation and Location

Where is FabriControl installed?

FabriControl is a desktop application that installs directly on a Windows PC. During installation, all files are copied to:

📁 Default location:
C:\Program Files\FabriControl\

System Requirements

Component Minimum Recommended
Operating System Windows 10 Windows 11
Processor Intel Core i3 or equivalent Intel Core i5 or higher
RAM 4 GB 8 GB or more
Hard Drive 10 GB free 50 GB free (for files)
Network Not required (local mode) LAN network for multiple users
⚠️ Important: FabriControl ALWAYS requires an internet connection for license validation when logging in, even in local mode.

File Structure

After installation, FabriControl creates the following structure:

C:\Program Files\FabriControl\
├── 📁 frontend/          → Web interface (React)
├── 📁 backend/           → Server (Python/FastAPI)
├── 📁 mongodb/           → Database (if local)
├── 📁 uploads/           → Uploaded files (PDFs, photos)
├── 📁 backups/           → Automatic backups
└── 📄 FabriControl.exe   → Main executable

Understanding "Desktop" vs "Cloud"

FabriControl is ONE installer that adapts to your business. During installation, you configure how you want to use it:

Aspect 💻 Local Mode 🌐 LAN Mode ☁️ Cloud Mode
Where? One PC only Server in your workshop Cloud server (e.g., AWS)
MongoDB On the same PC On the server MongoDB Atlas (cloud)
Users 1 person Multiple on local network Multiple from anywhere
Internet Required? Yes, for license validation Yes, for license validation Always (cloud)
Same installer? ✅ Yes, the SAME .exe adapts to your configuration

🗄️ 2. MongoDB Database

What is MongoDB?

MongoDB is the database where FabriControl stores all your business information:

  • 📋 Quotes and estimates
  • 📦 Material inventory
  • 👥 Customer and supplier data
  • 🏭 Production orders
  • 📊 Reports and statistics
💡 Simple Analogy:
Think of MongoDB as a super-organized digital filing cabinet where all your business documents are stored. It's like having all your folders, but in electronic form and much faster to search.

MongoDB Options

During installation, you can choose between 2 options:

Option What does it mean? Who is it for?
🏠 Local MongoDB Database is installed on your server PC. All data is physically on your computer. ✅ Small workshops
⚠️ Requires internet to validate license
✅ Maximum privacy
☁️ Cloud MongoDB Database is on the internet (MongoDB Atlas). Your data is on secure online servers. ✅ Multiple locations
✅ Remote access
✅ Automatic backups

Where is Your Data Stored?

🏠 If you chose Local MongoDB:
Data is stored at: C:\Program Files\FabriControl\mongodb\data\
Initial size: ~500 MB
Growth: +10-50 MB per month (depends on usage)
☁️ If you chose Cloud MongoDB:
Data is on MongoDB Atlas servers (AWS)
Physical location: Selectable (USA, Europe, Brazil, etc.)
Encryption: Yes, automatic

Database Size

Typical growth example:

  • Small workshop (50 orders/month): ~2 GB per year
  • Medium workshop (200 orders/month): ~8 GB per year
  • Large factory (1000+ orders/month): ~40 GB per year
⚠️ Important: Attached files (photos, PDFs) are NOT stored in MongoDB. They are stored separately in the uploads/ folder (see next section).

💾 5. Backups and Security

Automatic Backups

FabriControl automatically backs up your information:

📦 Automatic Backups:
• Frequency: Daily (at 2:00 AM)
• Location: C:\Program Files\FabriControl\backups\
• Retention: Last 7 days
• What's included? Complete database (MongoDB)

How to Make a Manual Backup?

From the administration panel:

  1. Go to Settings → Backups
  2. Click "Create Backup Now"
  3. Wait 1-5 minutes (depends on data size)
  4. Download the .backup file and save it in a safe place

How to Restore a Backup?

  1. Go to Settings → Backups → Restore
  2. Select the .backup file
  3. Confirm restoration
  4. The system will restart automatically
⚠️ Important:
• Attached files (photos, PDFs) are NOT included in the automatic MongoDB backup
• You must manually backup the uploads/ folder
• Recommendation: Copy the uploads/ folder to an external disk weekly

Data Security

  • 🔒 Encryption: All sensitive data is encrypted
  • 👤 Access control: Each user has their password and permissions
  • 📝 Activity log: Records who did what and when
  • 🔐 Authentication: JWT tokens for maximum security

Data Flow

User types in browser:
     │
     ▼
[FRONTEND] → Sends data to server
     │
     ▼
[BACKEND] → Processes and validates data
     │
     ▼
[MONGODB] → Saves information
     │
     ▼
[BACKEND] → Confirms save
     │
     ▼
[FRONTEND] → Shows "✅ Saved" message

Ports and Services

Service Port What is it for?

If you need to configure Windows firewall, you must allow ports 3000 and 8001 so other users on the network can access.

Monthly Renewal

Each month, before the renewal date:

  1. System sends automatic payment reminder
  2. Client makes monthly payment ($49 or $129)
  3. Support verifies payment received
  4. License status remains as "active"
  5. If client DOESN'T pay, status changes to "suspended"
⚠️ No Payment = No Access:
If monthly payment is not received, system access will be automatically blocked on the renewal date. All data remains secure and will be reactivated upon receiving payment.

Frequently Asked Questions about Licenses

❓ Can I use FabriControl without internet?
NO. FabriControl ALWAYS requires internet connection to validate your license at login. Without internet, you won't be able to access the system.
❓ What happens to my data if I don't pay?
Your data (quotes, orders, inventory) is 100% secure. It remains intact on your local server or in the cloud. When you reactivate your license, you can access everything as before.
❓ How do I cancel my subscription?
Simply stop paying and your license will change to "suspended" status. If you want to permanently delete your account and data, contact support.
❓ Can I reactivate after cancelling?
Yes, at any time. Make the corresponding monthly payment and support will reactivate your license. All your data will be available.

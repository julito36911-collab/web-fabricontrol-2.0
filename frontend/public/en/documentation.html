🗄️ 2. MongoDB Database

What is MongoDB?

MongoDB is the database where FabriControl stores all your business information:

  • 📋 Quotes and estimates
  • 📦 Material inventory
  • 👥 Customer and supplier data
  • 🏭 Production orders
  • 📊 Reports and statistics
💡 Simple Analogy:
Think of MongoDB as a super-organized digital filing cabinet where all your business documents are stored. It's like having all your folders, but in electronic form and much faster to search.

MongoDB Options

During installation, you can choose between 2 options:

Option What does it mean? Who is it for?
🏠 Local MongoDB Database is installed on your server PC. All data is physically on your computer. ✅ Small workshops
⚠️ Requires internet to validate license
✅ Maximum privacy
☁️ Cloud MongoDB Database is on the internet (MongoDB Atlas). Your data is on secure online servers. ✅ Multiple locations
✅ Remote access
✅ Automatic backups

Where is Your Data Stored?

🏠 If you chose Local MongoDB:
Data is stored at: C:\Program Files\FabriControl\mongodb\data\
Initial size: ~500 MB
Growth: +10-50 MB per month (depends on usage)
☁️ If you chose Cloud MongoDB:
Data is on MongoDB Atlas servers (AWS)
Physical location: Selectable (USA, Europe, Brazil, etc.)
Encryption: Yes, automatic

Database Size

Typical growth example:

  • Small workshop (50 orders/month): ~2 GB per year
  • Medium workshop (200 orders/month): ~8 GB per year
  • Large factory (1000+ orders/month): ~40 GB per year
⚠️ Important: Attached files (photos, PDFs) are NOT stored in MongoDB. They are stored separately in the uploads/ folder (see next section).

💾 5. Backups and Security

Automatic Backups

FabriControl automatically backs up your information:

📦 Automatic Backups:
• Frequency: Daily (at 2:00 AM)
• Location: C:\Program Files\FabriControl\backups\
• Retention: Last 7 days
• What's included? Complete database (MongoDB)

How to Make a Manual Backup?

From the administration panel:

  1. Go to Settings → Backups
  2. Click "Create Backup Now"
  3. Wait 1-5 minutes (depends on data size)
  4. Download the .backup file and save it in a safe place

How to Restore a Backup?

  1. Go to Settings → Backups → Restore
  2. Select the .backup file
  3. Confirm restoration
  4. The system will restart automatically
⚠️ Important:
• Attached files (photos, PDFs) are NOT included in the automatic MongoDB backup
• You must manually backup the uploads/ folder
• Recommendation: Copy the uploads/ folder to an external disk weekly

Data Security

  • 🔒 Encryption: All sensitive data is encrypted
  • 👤 Access control: Each user has their password and permissions
  • 📝 Activity log: Records who did what and when
  • 🔐 Authentication: JWT tokens for maximum security

     │

Yes, at any time. Make the corresponding monthly payment and support will reactivate your license. All your data will be available.
